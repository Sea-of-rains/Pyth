Выполнила: Николаева Александра</p>
Здание: Сравните время работы двух реализаций функции вычисления факториала:
<ul>
<li>рекурсивной,
<li>нерекурсивной (через цикл).</li></ul></p>
Для сравнения используйте модуль timeit, а для наглядности постройте график с помощью matplotlib, показывающий зависимость времени выполнения от входных данных.
Условия:</p>
<ul><li>Реализуйте две функции fact_recursive(n) и fact_iterative(n).</li>
<li>Для корректного сравнения используйте одинаковый набор чисел для тестирования.</li>
<li>Визуализируйте результаты (ось X — размер входного числа n, ось Y — время вычислений).</li>
<li>Сравните эффективность двух подходов.</li></ul></p>
